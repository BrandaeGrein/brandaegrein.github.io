Calculus in VR

Meant to bridge the gap between my previous career in education and my new career in software development, this was my first ever coding project. Originally designed to teach calculus through more fun and story driven gameplay, I learned a valuable lesson about scope.

This project was made using Unity and the SteamVR Assets Package. There were a set of "tutorial" levels, an introductory story scene that introduced the antagonist Lord Zaladros, and then the main lesson level. The project was sadly completed before actually getting to teach calculus, so I settled for Vector Magnitude instead.

Functional gameplay included things such as a pen that players could write with, a calculator to help them figure out calculations in VR, and teleportation. Originally, the game also had cats, but the cats proved difficult to work with and required far too much cat food for the allotted budget.
